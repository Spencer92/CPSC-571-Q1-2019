Creates a database in a way that would need to be accessed using Secure Multiparty Computation as described in Defining Privacy for Data Mining by Chris Clifton, Murat Kantarcioglu, and Jaideep Vaidya. 

This was meant to be one half of the program, with the other half, the ability to retrieve information, unable to be completed within the deadline. 

Tested and working on a i5 3570k system.
